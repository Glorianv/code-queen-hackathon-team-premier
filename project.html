Welcome to CODE QUEEN

About us: Educating The Children is a UK registered charity that focuses on empowering young women through education and training that increases their employability. In 2019, we launched Code Queen in Kampala which is an all-female bootcamp designed to train young women in coding and soft skills. Code Queen has successfully run 3 cohorts and graduated 70 students since the launch. Register with us today. Remember, it is totally free of charge!.
